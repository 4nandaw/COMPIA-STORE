# Páginas do Frontend

Esta pasta deve concentrar as páginas do frontend da COMPIA Store.

Sugestão de organização futura:

- `HomePage.tsx` — listagem inicial de produtos / destaques
- `ProductDetailsPage.tsx` — detalhes de um livro
- `CartPage.tsx` — carrinho de compras
- `CheckoutPage.tsx` — fluxo de finalização de compra

Cada página pode reutilizar componentes em `src/components`.
